Mis Proyectos

MinePac

Este fue un juego creado en el lengueje de c++ con ayuda de la librería de gráficos Allegro en su verisón 5.2. La finalidad de este juego es cruzar un laberinto con temática de Minecraft, adquiriendo coleccionables y superando obstáculos y enemigos para completar el nivel.

Sistema Gestor de Inventario

Este proyecto consistió en crear un sistema gestor para una tienda. Para su realización se uso el lenguaje de Java y con un sistema de bases de datos SQLite. El núcleo del proyecto fue un CRUD de los artículos del inventario, más la generación de reportes.

Sistema BS y RR

Este proyecto es un simulador de dos sistemas que funcionan simultaneamente para la liberación y asignación de memoria de forma similar en que lo hace un sistema operativo real. El lenguaje utilizado fue c++ usando librerías internas para el funcionamiento del entorno gráfico.

Juego de Sudoku

Este juego fue creado en el lueguaje de c. La finalidad del proyecto fue crear un juego de Sudoku junto con con otro llamada Da y Quita.

Piano en Ensamblador

Para este proyecto, se realizó un piano con el lenguaje Ensamblador. Se pueden reproducir canciones pregrabadas y jugar con un piano de manera interactiva.
